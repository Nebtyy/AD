**Checks for:**

- Active Directory Domain Info
- MachineAccountQuota
- Print Spooler on Domain Controllers (PrintNightmare risk)
- NTLM Compatibility Level
- SMB Signing Requirement
- LLMNR Status
- WPAD AutoConfig Status
- Kerberoastable Accounts (SPN users)
- AS-REP Roastable Accounts
- Unconstrained Delegation
- Constrained Delegation
- Resource-Based Constrained Delegation (RBCD)
- PasswordNotRequired Users
- AdminSDHolder Existence
- DC Sync Permissions (Replicating Directory Changes rights)
- ADCS Presence (Certificate Services Enrollment Services)
